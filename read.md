# Online Complaint Management System

A Spring Boot RESTful application to manage complaints. This project allows users to **create, view, update, and delete complaints** efficiently.

## Features
- Create new complaints via REST API
- Retrieve all complaints or a specific complaint by ID
- Update complaint status (OPEN, IN_PROGRESS, RESOLVED)
- Delete complaints
- Built with Spring Boot, JPA, and MySQL

---

## Technologies Used
- **Backend:** Java 21, Spring Boot 4, Spring Data JPA  
- **Database:** MySQL 8  
- **Build Tool:** Maven  
- **Testing:** Postman for API testing  
- **Security:** Spring Security (basic setup, can integrate JWT)
